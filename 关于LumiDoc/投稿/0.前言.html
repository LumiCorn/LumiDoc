    向 LumiDoc 投稿

    悠悠狮羝LumiCornX2025/11/24大约 2 分钟

    请在上传你的第一份文档前，仔细阅读该系列的所有内容，以防出现纰漏导致你的知识贡献被拖延。

    简介

    LumiDoc 作为文档托管网站，除了托管由 悠悠狮羝LumiCornX 亲手制作的文档外，也允许用户上传自己撰写的文档，以丰富 LumiDoc 的知识储备量。

    LumiDoc 的文档基于 Markdown 格式，所以在上传所有文档前，需要学会使用 Markdown 语法。此外，LumiDoc 还有独特的 FrontMatter 支持、独有小功能与第三方插件，将会在后文中全部提到。

    上传文档后，LumiDoc 将采用人工审核的方式检查文档是否符合所有要求，当所有要求全部满足时，文档将会被正确上传，并在上传后的 10 分钟左右发布到全球网络。

    接下来你可以根据标题顺序找到你想要的内容的快速定位：

    1. 学习 Markdown

    Markdown 是一种轻量级标记语言，排版语法简洁，让人们更多地关注内容本身而非排版。它使用易读易写的纯文本格式编写文档，可与 HTML 混编，可导出 HTML、PDF 以及本身的 .md 格式的文件。如果你是第一次听说 Markdown，你可以点击下面的链接来快速学习 Markdown 的语法：
