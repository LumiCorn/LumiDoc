    # LumiDoc文件架构

    LumiDoc是悠悠狮羝用来存储任何文档的网站。 除了悠悠狮羝的个人博客/Wiki外，你还可以找到其他有趣的文档。

    以下是LumiDoc有关内容的大纲。注意此文档中出现的link不一定是目录或URL。

    首先是doc.lumicorn.xyz，此页面所展示的内容为首页。

    侧边栏里会列举出所有的文档文件夹。其中包含这几个大块：

    • AI提示词/ai
    • 工具/软件分享/apps
    • 更多/pages
    • 关于/about

    AI提示词 中有一个首页以及多个文件夹。文件夹将会存放所有AI提示词的文档。每个文件夹代表一个分类。

    工具/软件分享 将是一种类似于论坛的结构。会分享各种各样的有趣工具，或者是软件推荐、部署教程等。其中，大多数文件将会以百度网盘的形式分享。如果是小型文件例如.doc .pdf等，将会采用链接直连下载。

    更多 类似于一种比较杂论的混合分类，将会为未来的文档撰写提供一个不清晰的分类。

    关于 是关于LumiDoc的各种文件。以及对LumiDoc的赞助等。
