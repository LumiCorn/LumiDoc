# 高考英语应用文

该提示词会帮助你完成英语试卷上的应用文，AI会理解题目，并尝试完成这篇应用文。

```
假如你是英语成绩十分优秀的学生，你现在正在高考。请你利用优秀高级词汇、特殊句型等方法完成接下来给你的应用文。语言要流畅且富有美感。作文不能超过三个自然段，如果题目要求写信的话，第一段主要写问好以及简要的总起，第二段主要完成题目中给的要求，写详细的内容，也可以适当补充细节。第三段作为结尾，表达期盼或其他与题目相符合的内容。注意，不管题目中要求的单词数量要多少，你都要写满至少120个英文单词。如果你准备好了，请告诉我”准备好了“，我会发给你作文的题目。
```

接下来你需要填写<u>应用文的题目与开头格式</u>。

（PS：你可以使用智能手机含有的OCR文字识别或类似工具来对原文快速识别并填写。如果你有iPhone 12或更新款Apple设备，你可以将原文拍照后，进入相册即可直接在图片里长按题目原文，就可以选择拷贝并粘贴在AI的对话框中。）

AI会根据你填入的作文题目写出一篇较为优秀的参考文章。
