LumiDoc 创建的理由本是为了分享一些好玩的程序，以及提供一些有意义的--保姆级--教程，并且也是给悠悠狮羝本人留下一个域名`lumicornx.xyz`，所以“顺便”创建了一个托管在 GitHub Pages 上的网站。

## LumiDoc 有什么？

悠悠狮羝会在 LumiDoc 上保存一些 Markdown 文件（你现在正在阅读的这些文字就存储在一个 Markdown 文件中），这些文件在 LumiDoc 里被称作 **文档**，目前为止，这些文档大概内容有：
- AI 提示词（专门用于--驯化--配置对话式生成 AI，以便你快速让 AI 生成你想要的内容）
- 程序/工具分享（分享一些好玩的，或者是能解决一部分痛点的程序，以及手把手教学的安装办法。）
- 随笔/小纸条（悠悠狮羝日常发疯的想法💡）
- 教程（主要是面向悠悠狮羝个人感兴趣的内容的教学，这些教学一般比较详细，并且有详细的截图教学。）

除此之外，LumiDoc 还会不断地加入更多有意思的内容！

## LumiDoc 的运作方式？

LumiDoc 基于 VuePress 运行，并且采用了开源的 vdoing 主题框架。LumiDoc 的服务运行在 GitHub Pages 中，所以并没有任何开销（除了域名的续费钱💰。）
