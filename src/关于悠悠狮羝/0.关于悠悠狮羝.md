接下来我都是想到什么说什么。你将会看到逆天流水账+个人独白。
👋你好，我是悠悠狮羝。在 B 站上我是一个搞笑游戏 up 主兼 VTuber，在 GitHub 上我是 toC 的开发者。~~在我看来，C 端编程才是最有意思的。~~
我最喜欢做的事就是玩游戏了。朋友们都说我这个人很会搞笑，建议我试试边玩游戏边拍视频。所以我从 0 开始学习 Premiere Pro。你可以从我最早的视频开始看，我真的是自己完全自学的，没有报任何剪视频的课，完全就是遇到一个问题就上网搜一下的。
2019 年那会我还是小学生（字面），那会拿着家里 i5-8400 + 1060 的电脑就开始学 Premiere 了。实在是太卡了，但我整整用了它 6 年。6 年里确实从陈睿手里挣了点钱。但我好像发现我挣到的钱都是和 Furry 有关的？我其实不是特别想成为福瑞博主吧，对我来说兽设只是一个形象，我也不想混进这个圈子里。
UP 主的生涯从小学一直当到大学也是很牛逼了好吧，我发现我遇到的事也太多了，遇到了各种各样的人，我感觉我的生活比他妈游戏刺激多了，神人太多了。
我会想到什么就加什么的。 2026.5.25
