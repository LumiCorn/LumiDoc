## 下载

请点击下面的两个链接，分别下载 **TeamSpeak 3** 与 **TeamSpeak 3 汉化包**。

**TeamSpeak 3**：

- Windows 端：[点击我就可以开始下载](https://files-services.teamspeak.app/releases/client/3.6.2/TeamSpeak3-Client-win64-3.6.2.exe)
- macOS 端：[点击我就可以开始下载](https://files-services.teamspeak.app/releases/client/3.6.2/TeamSpeak3-Client-macosx-3.6.2.dmg)
- Linux 端：[点击我就可以开始下载](https://files-services.teamspeak.app/releases/client/3.6.2/TeamSpeak3-Client-linux_amd64-3.6.2.run)

**TeamSpeak 3 汉化包**：

[点击我就可以开始下载](https://dl.tmspk.wiki/https:/github.com/VigorousPro/TS3-Translation_zh-CN/releases/download/snapshot/Chinese_Translation_zh-CN.ts3_translation)

下载完成后，你将会在下载的文件夹内发现一共两个文件：

<i class="fa-solid fa-file"></i> TeamSpeak3-Client-win64-3.6.2.exe
<i class="fa-solid fa-file"></i> Chinese_Translation_zh-CN.ts3_translation

![下载文件夹](/assets/image/TS3DL-TL/下载完成.png)

## 安装

双击 **<i class="fa-solid fa-file"></i> TeamSpeak3-Client-win64-3.6.2.exe** 并运行，进入安装界面后点击 Next> 即可继续。

![开始安装](/assets/image/TS3DL-TL/开始安装.png)

在 License Agreement 界面里，需要 **先将中间的条款拉到最下面**，才能 **勾选下方的「I accept the terms of the License Agreement」勾选框**。完成后再点击 Next> 继续安装。

![条款页](/assets/image/TS3DL-TL/条款页.png)

选择安装用户与安装路径可以根据自己的需要来选择。

![选用户](/assets/image/TS3DL-TL/选用户.png)

![设置安装路径](/assets/image/TS3DL-TL/安装路径.png)

**「Choose Configuration Location」** 指的是选择用户自定义设置的存储位置，**如果你不知道是什么的话**，直接 Next> 即可。

![设置存储路径](/assets/image/TS3DL-TL/设置存储路径.png)

**「Overwolf」** 是一个类似于开麦显示的插件，和 KOOK 语音的侧边游戏内显示功能很像。可以不安装。

![OVERWOLF](/assets/image/TS3DL-TL/OVERWOLF.png)

点击 Install 按钮后即可完成安装。完成后需要你先点开 TeamSpeak 一次，并将其关闭。

## 汉化

找到下载文件夹中的 <i class="fa-solid fa-file"></i> Chinese_Translation_zh-CN.ts3_translation，双击打开，然后点击安装即可。

![汉化包](/assets/image/TS3DL-TL/package_inst_jxGQ8JK9qg.png)

安装完成后，你就完成了 TeamSpeak 3 的安装与汉化。

## 须知

TeamSpeak 的服务器完全由私人搭建，所以在每次启动 TeamSpeak 时，都需要点击应用内左上角的 `连接(C)` 来连接到对应的服务器。
