# 简介

LumiDoc 作为文档托管网站，除了托管由 悠悠狮羝LumiCornX 亲手制作的文档外，也允许用户上传自己撰写的文档，以丰富 LumiDoc 的知识储备量。

LumiDoc 的文档基于 Markdown 格式，所以在上传所有文档前，需要学会使用 *Markdown 语法*。此外，LumiDoc 还有独特的 FrontMatter 支持、独有小功能与第三方插件，将会在后文中全部提到。

上传文档后，LumiDoc 将采用人工审核的方式检查文档是否符合所有要求，当所有要求全部满足时，文档将会被正确上传，并在上传后的 10 分钟左右发布到全球网络。
